# Healthy Recipe App (UI Prototype)

A modern Flutter-based mobile UI for discovering healthy meals and exploring diet-friendly recipes.

This project focuses on UI/Frontend only, showcasing the design and user experience of a healthy recipe application before backend and business logic are implemented. Currently, the app is a frontend-only prototype with no backend integration.

## Features
- Modern and clean Flutter UI
- Recipe search interface
- Food category browsing
- Diet recommendation section
- Popular meals showcase
- Reusable UI components

## Tech Stack
- Flutter
- Dart
- SVG Asset Integration
